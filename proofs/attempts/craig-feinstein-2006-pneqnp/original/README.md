# Original Source Notes

This directory reconstructs the source material for Craig Alan Feinstein's
2006 article "A New and Elegant Argument that P is not NP", listed as entry #31
on Woeginger's P-versus-NP page.

The repository currently records the proof idea in `ORIGINAL.md` rather than a
verbatim source PDF. The reconstruction is intentionally conservative: it
captures the argument pattern needed for formal analysis without treating the
informal prose as an accepted proof.
