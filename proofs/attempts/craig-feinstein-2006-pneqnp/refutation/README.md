# Refutation

Feinstein's 2006 argument fails because it treats a counting fact about many
candidate witnesses as a lower bound against every polynomial-time algorithm.

Counting witnesses can show that exhaustive search is expensive. It does not
show that all algorithms must search exhaustively. A valid `P != NP` proof would
need to rule out every polynomial-time method for the NP-complete language, not
only direct inspection of exponentially many candidates.
