# Refutation Formalization

The refutation files isolate why Romanov's argument does not prove P = NP.

The paper relies on the implication from non-empty local SEP state to a global
joint satisfying assignment. The formal model separates local consistency from
global satisfiability and records that local consistency alone is insufficient.

The key lesson is that a polynomial bound for SEP would only prove P = NP after
a separate proof that SEP is globally sound and complete for 3-SAT. Romanov's
paper does not supply that proof.
