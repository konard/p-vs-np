# Original Source

This folder contains the source material for Koji Kobayashi's 2011 attempt.

- `ORIGINAL.pdf`: downloaded from arXiv:1110.0200.
- `ORIGINAL.md`: Markdown reconstruction of the English text from the arXiv
  source, with the original argument order preserved and commentary limited to
  short reconstruction notes.

The arXiv abstract page describes the paper as submitted on 2011-10-02 and last
revised on 2011-11-21. The paper title is "NP is not AL and P is not NC is not
NL is not L".
