## Overview

This folder reconstructs the February 2012 attempt `P versus UP`, later archived on arXiv as `1205.3655` and withdrawn because it used a pseudonym.

The paper's intended argument is:

1. Assume `P = UP`.
2. Claim that this forces `EXP = P`.
3. Use the Time Hierarchy Theorem (`EXP ≠ P`) to obtain a contradiction.
4. Conclude `P ≠ UP`.
5. Since `UP ⊆ NP`, claim `P ≠ NP`.

The formal refutation in [`../refutation/README.md`](../refutation/README.md) explains why the implication from `P = UP` to `EXP = P` is not justified and why `P ≠ UP` still does not imply `P ≠ NP`.
