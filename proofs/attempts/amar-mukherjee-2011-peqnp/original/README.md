# Amar Mukherjee (2011) - Original Proof Idea

## Overview

Amar Mukherjee's withdrawn 2011 arXiv preprint, "The 3-satisfiability problem",
claimed a deterministic polynomial-time algorithm for 3-SAT. Since 3-SAT is
NP-complete, such an algorithm would imply P = NP.

The full withdrawn paper is no longer available from arXiv, so this folder
contains the best available markdown reconstruction from the public metadata,
abstract, withdrawal notice, and the Woeginger list entry.
